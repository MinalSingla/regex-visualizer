# RegexAuto

An interactive web application that converts **Regular Expressions** into **ε-NFA**, **DFA**, and **Minimized DFA** with step-by-step visualization.

## Features

- Thompson Construction
- Subset Construction
- Hopcroft DFA Minimization
- Regex parsing
- ε-NFA construction
- DFA conversion
- Minimized DFA generation
- String simulation
- Interactive SVG visualization
- Zoom & Pan
- Step-by-step animations

## Technologies

- HTML
- CSS
- JavaScript
- React 18 (CDN)
- SVG
